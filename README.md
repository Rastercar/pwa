# Rastercar PWA

The worlds best car tracking service :car: :blue_car: :taxi: :bus:

```bash
# Install the dependencies
yarn

# Start the app in development mode (hot-code reloading, error reporting, etc.)
quasar dev

# Lint the files
yarn run lint

# Build the app for production
quasar build
```
